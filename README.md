# Shopping Cart Unit Testing Project

## Project Overview

This JavaScript project implements and tests basic shopping-cart
functionality, including adding items, removing items, and calculating
the total number of items.

## Functional Requirements

- Add items to the cart
- Remove a selected item
- Return the total number of items
- Preserve unaffected items after a removal
- Return zero when the cart is empty

## Functions

### addItem(item)

Adds an item to the shopping cart.

### removeItem(item)

Removes the specified item from the shopping cart.

### getTotalItems()

Returns the number of items currently in the cart.

## Testing

Unit tests validate expected behaviors, including:

- Adding one or more items
- Removing an existing item
- Preserving remaining items
- Counting populated and empty carts
